# KPI Format Rules — DataTech Vietnam

## Thresholds
- Revenue drop > 10%: flag as "NEEDS ATTENTION" (red)
- Revenue growth > 10%: flag as "STRONG GROWTH" (green)
- Change within +/- 10%: flag as "OK" (neutral)

## Currency
- Always display in VND with thousand separators
- Use format: 1,234,567,890

## Regions
- Valid regions: Hanoi, HCMC, Da Nang
- Always include all regions unless user specifies otherwise
- Sort by revenue (highest first)

## Period Format
- Use YYYY-MM format for month references
- Always show both current and prior period in the header

## Executive Summary Rules
- Maximum 3 sentences
- Always mention the best-performing region
- Always mention any region needing attention
- Include specific percentage numbers
